## Calculation Logic / ркЧркгркдрк░рлА рк▓рлЛркЬрк┐ркХ

### Formula / рк╕рлВркдрлНрк░:
```
Subtotal = (Quantity ├Ч Price) - ((Quantity ├Ч Price) ├Ч Discount / 100)
Tax = Subtotal ├Ч 0.18 (18%)
Total = Subtotal + Tax
```

### Example / ркЙркжрк╛рк╣рк░ркг:
- Quantity: 2
- Price: 100
- Discount: 10%
- Calculation: (2 ├Ч 100) - ((2 ├Ч 100) ├Ч 10 / 100) = 200 - 20 = 180
- Tax: 180 ├Ч 0.18 = 32.4
- Total: 180 + 32.4 = 212.4
